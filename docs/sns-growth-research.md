# 調査: 見られていない状態から、どう売れるようになったか

作成: 2026-07-17。[twitter-strategy.md](twitter-strategy.md)の「有料化」節を、この調査で一部**訂正**した(§5参照)。

---

## 1. 最重要の発見: フォロワー数と売上は、ほぼ無関係

| 事例 | フォロワー | 結果 |
|---|---|---|
| 有料note研究家 | **13人**(開設3日目) | 初売上。3ヶ月で累計350部 |
| 別の販売者 | **76人** | 60部購入 |
| 別の販売者 | **17人** | 初日に売れた |
| 逆の例 | **5,000〜6,000人** | 購入者5人 |

noteの公式データでは**有料記事の購入率は平均2〜3%**。そして繰り返し出てくる指摘が、
**「購入者の大多数はフォロワー以外」**(検索・SNS経由でたどり着いた新規)というもの。

⚠️ **ただしこの手の情報は割り引いて読む必要がある。** 「フォロワー13人で売れた」と発信しているのは
たいてい**"noteの売り方"を売っている人**で、生存者バイアスと利益相反がある。「売れなかった大多数」は
そもそも記事を書かないので観測できない。

**それでも構造的に納得できる理由がある:** 相互フォローで集めたフォロワーは買わない。
フォロワー数は「買う理由」ではないから。買う理由は**信頼**であって、規模ではない。

## 2. 逆の実例(失敗)— これが一番参考になる

[競馬note販売で失敗した主婦の告白](https://note.com/garigiri_ai01/n/ndedf5a889f29):

- **300円 × 3部 = 売上900円**。作業は毎週末3〜4時間 → **時給約30円**
- Xは順調だった(**1週間でフォロワー200人増、週末1,000〜3,000imp**)。**集客はできていたのに売れなかった**
- 失敗の原因として本人が挙げているもの:
  1. 300円は安すぎた(安さは価値の疑いを生む)
  2. 「毎週予想を届けます」の特典が持続不能になった
  3. **手作業の予想がAIの精度・速度に勝てなかった**
  4. 過去の経験に頼り、現在進行形の情報でなかった

**注目すべき点: この人が負けた相手のポジションに、あなたが立っている。** 教訓の4番目
「AI時代の競争戦略」は、まさにあなたのアプリのことを言っている。そして
「集客できていたのに売れなかった」という事実は、§1の結論(規模ではなく信頼が買われる)を裏から裏付ける。

## 3. 実際に回っている規模感(M&A市場の開示数字)

noteアカウントの売買案件では売主が実売上を開示する:

- 競艇予想note: **累計売上100万円以上、直近でも月平均8万円前後**で安定
- 競馬予想note: **総売上約38万円**

「月に数万円〜数十万円」が現実的な上限帯で、一発当てる世界ではない。
手数料は**決済5% + プラットフォーム10%**(売上の約15%が引かれる)。
価格相場は**300〜900円/レース**。

## 4. 0→1000フォロワーの実務(集客側)

[X運用6ヶ月で0→1万](https://note.com/taishiyade/n/n406400752f09)の初期戦術:

- **0〜100人の段階は「コバンザメ戦略」**: 大手アカウントのリプ欄に張り付く。フォロワー0のアカウントが
  自力でインプレッションを得る方法は他にほぼ無い
- 狙う相手の条件: **フォロワー1万人以上 かつ 直近の投稿が平均1万imp以上**(死んでいるアカウントは無意味)
- **通知をオンにして即リプし、リプ欄の上位を取る**
- プロフィールを「訪問者をフォロワーに変換する」前提で最適化する

アルゴリズム側の裏付け:
- **リプが付くとスレッド全体のimpが13.5〜75倍**に増える事例がある
- 2026年のXは「まず一部のフォロワーに流し、その反応率で拡散量を決める」構造。
  10本流して全部スルーされるより、1本でリプ・RTが集まる方が次の配信量が増える

⚠️ ただし**中身のないリプ(絵文字だけ等)は低品質と判定されて逆効果**。
「データで一言添える」うちの持ち味と噛み合う形でしかやらない。

## 5. 戦略の訂正(前回の提案を1つ撤回する)

前回`twitter-strategy.md`で有料化のゲートを **「n≥50 かつ ROI>100% かつ フォロワー1,000〜」** と書いたが、
**フォロワー1,000という条件はデータ的な根拠が薄いので外す。**

理由: §1・§2の通り、フォロワー数は売上をほとんど説明しない。むしろ「1,000人まで売らない」と決めると、
テストと改善の機会を何ヶ月も先送りすることになる(青山氏の指摘: 早く売った方が
タイトル・導入の改善サイクルが回る)。

**新しいゲート:**
- **実績: n≥50 かつ 対象セグメントのROI>100%** ← これは維持(これは集客の条件ではなく**信頼の条件**)
- **導線: note無料記事が検索・SNSから読まれている**(=フォロワー以外の流入がある)
- フォロワー数は条件にしない

## 6. この調査を踏まえた、うちの勝ち筋

失敗事例と自分たちの差分を並べると、やるべきことが自動的に決まる:

| 失敗事例 | うち |
|---|---|
| 手作業の予想(AIに負けた) | AIが全レース診断(**負けた側ではなく勝った側**) |
| 週3〜4時間の作業が続かず破綻 | **完全自動化済み**(作業時間ゼロ) |
| 過去の経験ベース、鮮度なし | 毎週の現在進行形+自動集計の実績 |
| 300円で安売り | 実績が出るまで**売らない** |

**具体的な打ち手(優先順):**

1. **X: 0→100はコバンザメ**。大手競馬垢(フォロワー1万+・直近1万imp+)の投稿に、
   **うちのデータで一言添えるリプ**を即座に返す。「そのレース、うちのAIはこう見てます(画像)」は
   宣伝ではなく貢献になり、うちの画像がそのままリプ欄で名刺になる
2. **note無料記事を検索の受け皿にする**。§1の「購入者の大多数はフォロワー以外」に賭ける。
   週次で「AIの全レース診断と実績」を無料公開し、レース名で検索した人が着地する場所を作る
3. **有料化は実績ゲートのみ。** 売るのは「本気診断(premium/Opus)を実行したレース」の
   買い目全体+フル分析。価格は相場上限寄り(**500〜900円**)。300円は安さが逆効果になる実例がある
4. **持続不能な約束を絶対にしない**(失敗事例の主因)。うちは自動化済みなので、
   「毎週必ず出す」は約束できる数少ないアカウント。**それ自体が差別化として言える**
